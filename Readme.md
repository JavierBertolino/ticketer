# Web

### Install dependencies
```bash
pnpm install  
```

### Start client
Add .env with 
```bash
VITE_API_URL
VITE_API_KEY # Not necessary in local environment
```

```bash
pnpm run dev
```



# Server

### Install dependencies  
  ```bash
  pnpm install
  ```

### Start server
Add .env with
```bash
EMAIL_ADDRESS
EMAIL_PASSWORD
PWD_HASH
JWT_SECRET
  ```


```bash
pnpm start
```
